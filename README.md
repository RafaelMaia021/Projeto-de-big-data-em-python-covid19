# Projeto Universitário de Big Data em Python - COVID19

## 🎓 Contexto Acadêmico
Este repositório faz parte de um **projeto universitário** desenvolvido na área de **Big Data aplicado à saúde pública**, com foco em análise de dados relacionados à COVID-19.  
O trabalho foi realizado como atividade acadêmica, utilizando ferramentas modernas de ciência de dados para explorar informações oficiais e apoiar a compreensão dos impactos da pandemia.

## 📌 Objetivo
Analisar dados de mortalidade e decisões regulatórias da **CONITEC**, aplicando técnicas de **Big Data** com **Python**, **Pandas** e **PySpark**.

## 📊 Escopo da Análise
- Evolução da COVID-19 em 2020.
- Comparação da mortalidade entre 2019 e 2020.
- Uso de visualizações para identificar tendências e apoiar decisões em saúde pública.

## 🛠️ Tecnologias Utilizadas
- Python
- Pandas
- PySpark
- GitHub para versionamento e colaboração

## 📈 Conclusões
- Crescimento expressivo dos registros de COVID-19.
- Mortalidade total em 2020 superou a de 2019.
- Visualizações facilitaram a identificação de tendências e apoiaram decisões em saúde pública.
